make & Makefile

Makefile 是一个指导性的文件（文件名就为 Makefile，无后缀）。

target 是构建的目标，可以是真的目标文件、可执行文件，也可以是一个标签。dependencies 是构建所需的其它文件或目标。之后的是构建所需的指令，每个 command 前必须有一个 tab，不能用空格。

提示

Makefile 会自动找到所有需要构建的文件进行操作。首先它会构建 all，然后发现需要构建 hello，然后执行 gcc -o hello hello.c。

之后执行 make hello 或是 make，即可产生 hello 这个可执行文件，使用 ./hello 就可以执行它。值得注意的是，当执行 make 指令而不附加其他参数时，默认构建第一个目标。

tmux

终端复用。

  • 输入 tmux 进入 tmux 的新会话。

  • ⌃ Ctrl + B 然后 ⇧ Shift + Num 5，将窗口左右分屏。
  • ⌃ Ctrl + B 然后 ⇧ Shift + 单引号，将窗口上下分屏。
  • ⌃ Ctrl + B 然后 ⬆ Up / ⬇ Down / ⬅ Left / ➡︎ Right，根据方向按键来切换窗格。
  • ⌃ Ctrl + B 然后 ␣ Space，切换窗格布局（上下变左右，左右变上下）。
  • ⌃ Ctrl + B 然后 X，关闭使用的窗格。
  • ⌃ Ctrl + B 然后 D，分离（Detach）当前会话，回到 Shell。原有的程序会保持在 tmux 会话中的状态。

  • 回到会话：在 Shell 下用 tmux ls 查看当前的会话列表，记住会话名（冒号左边的数字），使用 tmux a -t 会话名 恢复会话。
